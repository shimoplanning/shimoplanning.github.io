Privacy Policy

Simple Health Check does not collect, store, or share personal information.

This application performs HTTPS requests only to endpoints configured by the user for health check purposes.

Information Stored on Device

The following information may be stored locally on the user's device:

  • Registered site settings
  • Health check status
  • Response metadata such as HTTP status, response time, and Content-Type

This data is stored only on the user's device and is not transmitted to the developer or any third party.

No Personal Data Collection

This application does not:

  • Require user accounts
  • Use analytics services
  • Display advertisements
  • Collect personal data
  • Share user data with external services

Network Communication

Network communication is performed only for user-configured health check targets.

Policy Updates

If future versions introduce features involving data collection or external services, this policy may be updated accordingly.
